Guide d'Installation et de Mise à Jour

Découvrez comment installer et mettre à jour le logiciel MagPro en toute simplicité.

Nouvelle Installation

Lors de la première installation, le setup vous offre les choix suivants :

1. Mode du Programme :

  • Gestion de Stock : Dédié aux commerces et points de vente.
  • Restaurant : Optimisé pour la gestion des restaurants et cafés.

2. Type d'installation :

  • Serveur : Installe la base de données Firebird et les fichiers centraux sur le poste principal.
  • Client : Connecte le poste secondaire au serveur via son adresse IP.

Mise à Jour (Update)

En choisissant 'Mise à jour', le programme d'installation va :

  • Sauvegarder en toute sécurité votre fichier de configuration et votre base de données.
  • Nettoyer les anciens fichiers et déployer la nouvelle version automatiquement.
  • Aucune reconfiguration réseau ou serveur n'est requise.
